# Project Rules

- **No Duplicates**: Never duplicate or repeat texts, titles, or images across ebooks.
- **Synthesis Accuracy**: Ensure every "Síntese" (description/summary) corresponds accurately to the author and the specific work.
- **Typography**: 
    - Do NOT use italics for body text in ebooks.
    - Headers and special accents can use specific styling, but the main reading content must be normal weight/style for better legibility.
- **Navigation**:
    - Always provide a "Voltar" (Back) button to the previous page.
    - When navigating between chapters/pages, automatically scroll to the top of the content.
- **Performance**:
    - For the primary admin (analista.ericksilva@gmail.com), bypass "Pending" state and "Loading" synchronization screens where possible to ensure immediate access.
- **Image Handling**:
    - Always use `referrerPolicy="no-referrer"` for external images (Unsplash, etc.).

# Aesthetics
- Follow the "Studio Logos" premium brand identity: Navy, Gold, and Off-white palette.
- High-contrast typography with serif fonts for titles (Cormorant Garamond) and sans-serif for UI (Inter).
